# 🎨 Aashu Portfolio

A modern, responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript. Showcase your skills, projects, and get in touch with visitors through an elegant and interactive interface.

![Portfolio Preview](https://img.shields.io/badge/Status-Active-brightgreen) ![License](https://img.shields.io/badge/License-MIT-blue)

## ✨ Features

- **Responsive Design** - Fully responsive layout that works seamlessly on all devices (mobile, tablet, desktop)
- **Mobile-Friendly Navigation** - Hamburger menu for mobile devices with smooth animations
- **Hero Section** - Eye-catching intro section with call-to-action button
- **About Section** - Professional introduction with skills showcase
- **Projects Section** - Display your projects and accomplishments
- **Contact Form** - Interactive contact form with validation and success feedback
- **Smooth Scrolling** - Smooth page navigation with anchor links
- **Modern UI** - Clean, professional design with gradient colors and shadows
- **Dark/Light Theme Ready** - CSS variables for easy theme customization

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS variables, flexbox, and grid
- **JavaScript (Vanilla)** - Lightweight, no dependencies required

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (VS Code, Sublime Text, etc.)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/aashutoshborse-coder/aashu.git
   cd aashu
   ```

2. **Open in Browser**
   ```bash
   # Simply open index.html in your browser
   # Or use a local server for better experience:
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

## 📖 Usage

### Customize Your Portfolio

1. **Update Personal Info** - Edit the text in `index.html` to add your name, bio, and details
2. **Modify Colors** - Change CSS variables in `styles.css`:
   ```css
   :root {
       --primary-color: #667eea;
       --secondary-color: #764ba2;
   }
   ```
3. **Add Your Projects** - Add new project cards in the projects section
4. **Customize Skills** - Update your skills list in the about section

### Sections

- **Navigation** - Links to Home, About, Projects, and Contact
- **Hero** - Main introduction with professional title and CTA button
- **About** - Your background and skills
- **Projects** - Showcase of your work
- **Contact** - Get messages from interested visitors

## 🎯 Features Explained

### Responsive Navigation
- Hamburger menu on mobile devices
- Smooth menu transitions
- Auto-closes when clicking on links or outside

### Contact Form
- Form validation
- Success feedback message
- Auto-resets after submission
- Non-functional (client-side only) - integrate with backend for actual email sending

### Smooth Scrolling
- Click navigation links for smooth page navigation
- CSS scroll-behavior for elegant transitions

## 🔧 Customization

### Add Your Own Styles
Edit `styles.css` to match your personal brand:
- Change primary colors
- Adjust spacing and padding
- Modify font families
- Update animations and transitions

### Enhance JavaScript
Extend `script.js` to add:
- Form validation and email integration
- Animated counters
- Dark mode toggle
- Additional interactive features
